# OptimizacijskiAlat

## Instalacija

Kako bi se rjesenje koristilo, potrebno je:

1. Kroz Visual Studio NuGet Package Manager preuzeti posljednju stabilnu verziju Google OR-Tools
2. Preuzeti instalaciju or-tools na:
3. dodati put do \or-tools_VisualStudio2019-64bit_v8.0.8283\include u Additional Include Directiories u Project Settings -> C/C++ -> 
4. dodati put do \or-tools_VisualStudio2019-64bit_v8.0.8283\lib u Additional Library Dependencies u Project Settings -> Linker -> General
5. dodati ortools.lib u Additional Dependencies u Project Settings -> Linker -> Input
